# Release Notes

## v1.0.0 - Initial Release

We're excited to launch the Retro Calculator, a simple yet powerful web-based calculator application!

### Key Features
- **Basic Arithmetic Operations**: Perform addition, subtraction, multiplication, division, and percentage calculations.
- **Calculation History**: View the full expression before the result.
- **Keyboard Support**: Use keyboard inputs for numbers, operators, equals, backspace, and escape for enhanced accessibility.
- **Chain Calculations**: Continue calculations using the previous result.
- **User-Friendly Controls**: Includes clear (AC) and delete (DE) buttons for easy operation.
- **Error Handling**: Gracefully manages invalid calculations.

Built with HTML, CSS, and JavaScript for a clean, responsive experience. Open `calculator.html` in any web browser to get started!

For more details, check the [README.md](README.md).
